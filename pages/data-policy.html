편집·출처 정책

본 사이트는 사용자가 정보를 검증할 수 있도록 출처를 명시하고, 요약 정보와 원문 링크를 함께 제공합니다. 본문 해설은 자동 생성 요약이 아닌 운영자의 수동 편집 문장을 원칙으로 합니다.

출처 선택 기준

  • 고양시청, 경기도 공공기관, 국가통계포털 등 공신력 있는 1차 출처 우선
  • 실시간 데이터는 공개 API(Open-Meteo)에서 제공되는 수치만 표시
  • 확인되지 않은 커뮤니티 글, 익명 게시물, 출처 불명 자료는 배제

콘텐츠 작성 기준

  • 중복 문서 복제 대신 고양시 사용자 관점의 설명을 새로 작성
  • 광고 클릭을 유도하거나 오해를 일으킬 문구 사용 금지
  • 정책·통계 수치 언급 시 기준 시점과 출처 링크 병기

업데이트 정책

실시간 카드 영역은 페이지 로드 시 자동 갱신됩니다. 정적 본문/정책 문서는 분기별 점검을 원칙으로 하며, 주요 출처 구조 변경 또는 정책 변경이 확인되면 즉시 개정합니다.
