데이터 출처와 편집 원칙

주민공람 레이더는 공고를 직접 생산하는 기관이 아니라, 주민이 원문 공고를 더 쉽게 찾고 읽도록 돕는 독립 정보 서비스입니다. 따라서 데이터 출처와 편집 원칙을 명확히 공개하는 것이 중요합니다. 홈과 상세 페이지의 표기 정보는 1차 출처를 기준으로 정리하며, 실제 제출 전에는 반드시 원문 공고문과 첨부문서를 다시 확인해야 합니다.

주요 데이터 출처

  • 시·군·구 고시공고 게시판
  • 광역지자체 도시계획·도시공간 관련 게시판
  • 토지이음, 도시계획도, 공공 지도 서비스 등 위치 확인용 자료
  • 관계 기관 대표 페이지 및 열람 장소 안내 문서

AI 요약 원칙

  • 행정문서를 일반 시민이 이해하기 쉬운 문장으로 바꾸되, 원문에 없는 사실을 단정하지 않습니다.
  • 찬반을 유도하지 않으며, 정치적 표현을 사용하지 않습니다.
  • 무엇이 바뀌는지, 왜 의견을 받는지, 누가 특히 봐야 하는지, 언제까지 무엇을 해야 하는지를 중심으로 정리합니다.
  • 요약 하단 또는 상세 설명에는 원문 기준 최종 확인 필요 문구를 유지합니다.

원문 우선 원칙

주민공람 레이더의 텍스트, 카드, 지도, 상태 배지는 탐색을 돕는 요약 인터페이스입니다. 법적 효력과 최종 기준은 각 기관이 게시한 원문 공고문, 첨부도서, 담당 부서 안내에 있습니다. 특히 마감 시각, 제출처 주소, 첨부도서 버전은 변동될 수 있으므로 제출 직전 재확인이 필요합니다.

오류 제보 안내

링크 오류, 기간 오기, 좌표 오차, 설명 누락 등은 문의 페이지를 통해 제보할 수 있습니다. 현재 공개된 데이터셋은 수도권 시범 구조이며, 전국 확장 전에 데이터 검증 흐름과 UI 사용성을 먼저 점검하고 있습니다.
